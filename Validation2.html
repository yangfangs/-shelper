Shelper Help Document

1. Select Genome Version:

Select the reference genome version you are working with. You can choose between hg19 and hg38. This will determine the reference for the variant locations you input.

2. Input Validate Location and File Name:

Enter the genomic location and the corresponding file name, separated by a comma. Each entry should be on a new line. The genomic location should be in the format of 'chromosome-position-reference>alternative', and the file name should correspond to the chromatogram file you intend to upload.

Example:

17-41245245-CT-C,example1.ab1
13-32893229-G-A,example2.ab1
13-32914859-G-GA,example3.ab1
13-20763530-CACACGTTCTTGCAGCC-C,example4.ab1
13-20763210-CGTT-CGTTCGTT,example5.ab1

3. Upload Chromatogram File:

Upload your chromatogram files here. The tool accepts files in .ab1 or .scf format. You can upload multiple files at once by selecting them in the file dialog or dragging and dropping them into the upload area.

Example (The uploaded .ab1 file must be the same as the entered File Name):

example1.ab1
example2.ab1
example3.ab1
example4.ab1
example5.ab1

4. Load Example Data:

If you want to test the tool or see how it works, you can load example data by checking this box. This will populate the input fields and generate a sample output based on preset example data.

5. Submit:

After completing the above steps, click the 'Submit' button to process the data. The results will be displayed in the 'Results' tab, including the variant location, genotype, and an option to view the chromatogram plot.

Results Tab:

The 'Results' tab will show the Check Result Table, where you can review the data associated with each variant location you've entered. It displays location, genotype, a checkmark indicating successful processing, and the start and end positions on the reference genome. You can also click 'Show' to view the chromatogram plot for each location.

Chromatogram Plot:

Below the table, you will find the chromatogram plot which visualizes the sequencing data. You can use this to visually inspect the variants.

Alignment Information:

At the bottom of the page, detailed information about the alignment is provided, including the Alignment, Reference Sequence, Alternate Allele, and Alignment Header.
